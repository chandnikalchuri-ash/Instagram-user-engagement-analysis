# Instagram-user-engagement-analysis
SQL-based analysis of Instagram user engagement, activity levels, and marketing insights.

---

## 📌 Problem Statement

Analyze Instagram user data to understand engagement patterns, identify high-value users, and generate business insights for marketing strategies.

---

## 📂 Dataset

The dataset includes:
- Users  
- Photos  
- Likes  
- Comments  
- Follows  
- Tags  
- Photo_Tags  

---

## ⚙️ Approach

- Performed data cleaning (NULL & duplicate checks)  
- Used SQL joins and aggregations  
- Applied CTEs and window functions  
- Calculated engagement metrics  
- Segmented users based on activity  

---

## 🏗️ Architecture Overview

The project follows a structured data pipeline:

- Data Source: Users, Photos, Likes, Comments, Follows, Tags
- Data Cleaning: NULL & duplicate checks
- Processing Layer: SQL joins, aggregations, CTEs, window functions
- Analysis Layer: Engagement metrics, user segmentation
- Output: Insights, charts, and business recommendations


---

## 📊 Key Insights

- 26% users are inactive → retention opportunity  
- Top 15–20% users drive most engagement  
- Consistent posting increases engagement  
- Emerging users = growth potential
